# Unexpected Code Execution (ASI05)

> OWASP Agentic 2026 — agent-generated or fetched code runs without sandbox/review.

## Agent one-liners

- Never execute code fetched from external URL without review.
- Sandboxed test runs for untrusted snippets — not on host with prod credentials.
- No `eval`, `exec`, dynamic import of user/agent-generated strings in app code.
- Agent-suggested scripts require human review before run in privileged env.

## MUST

- CI runs tests in isolated runner — not agent shell with secrets.
- Generated code reviewed like human PR before merge to main.
- Dependency install only from lockfile — not agent-suggested package names without verify.

## MUST NOT

- Pipe agent output directly to `bash` without inspection.
- Auto-run migration/SQL generated by agent on production.

## Maps to

- `../injection-prevention.md`, `../software-data-integrity.md`

## Agent action

Run generated code in least-privilege sandbox first; show diff before execute.
